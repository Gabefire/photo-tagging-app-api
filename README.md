## Getting Started

### Installing and running

```
git clone https://github.com/Gabefire/photo-tagging-app.git
cd photo-tag-app
npm install
npm run start
```

### Built With

- TypeScript
- Express
- MongoDB

## How to Play?

- Either log in, sign up or press try me to start
- Click play on "how to" screen
- Find the characters as fast as possible
